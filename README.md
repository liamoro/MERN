# MERN_URL_Shortener

MERN project 1.0
Сервис для сокращения ссылки.

После авторизации, пользователь может создать базу своих сокращенных ссылок.
Ссылки доступны только создателю.
Для пользователя доступна инф о ссылке, дате создания и количеству переходов на нее.

Аторизация реализована на основе jwt.

запуск приложения:
 --> npm run dev


(Добавлены файлы для поднятия проекта в Docker)
Makefile для удобного запуска.
--> make run-dev
